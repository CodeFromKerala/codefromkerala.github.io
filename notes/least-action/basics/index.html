What is Action ?

“Principle of least action”, so we should know what action is.

Action is just a mathematical quantity - which is analogous to energy but not exactly energy. We usually imagine an object to have energy. Similarily, we imagine the path a object follows to have specific action.

For example, a spaceship simply falling vs moving in a zig-zag fashion have different paths, with different action.

Now, action is just a number like energy, don’t get me wrong.

Imagine a ball of mass $m$ moving with some velocity $v$ and achieving displacement of $s$. Here’s a bit of math so we have formalism.

$$ S = mvs $$

Here S is action for the mentioned path. $m, v, s$ have their previously mentioned meanings here.

Principle Of Least Action

Well, the principle of least action means exactly that. If there are a bunch of paths a object can take, the object takes the path with the least action.
